== Description ==
显示前后台是否回复时邮件通知选项，内置回复邮件模板。开发这个插件的目的是为了修改留言email地址不正确的时候回复发邮件的后台修改成不发邮件


Related Links:
* <a href="http://www.phpsong.com">Plugin Homepage</a>

1. 上传 解压目录到 `/wp-content/plugins/` 目录
